# Brand System

## Colors

Use a neutral premium palette:

- Background: off-white or near-black
- Text: charcoal / white
- Accent: warm beige, champagne, or muted gold
- Borders: subtle gray

## Typography

Use elegant typography:

- Headings: serif or refined display font
- Body: clean sans-serif
- UI labels: readable sans-serif

## Layout

- Spacious
- Grid-based
- Image-first
- Responsive
- Minimal chrome

## Components

- Gallery grid
- Photo modal
- Selection button
- Download button
- Photographer dashboard
- Upload area
- Client access page

## Interaction

- Smooth hover states
- Clear selected state
- Keyboard accessible
- Mobile touch friendly
